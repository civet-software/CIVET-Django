CIVET:
Contentious Incident Variable Entry Template


System Features

  • We are currently developing this based on a fixed frame that is 960 pixels wide and 700 pixels high: this will fit easily on a 1024x768 screen. This is probably quite conservative—see this link for screen sizes being used by visitors to a popular development web site—but it is not unheard of for data labs to have older equipment so we're going with this at the moment
  • The prototype is currently deployed in the cloud on the “Google App Engine”; we currently expect the operational version will be deployed on a UNC server.
  • The system is written using the Flask framework, a Python-based system which is widely available on various cloud platforms.
  • The complete CIVET code will be licensed onto one of the open source licenses, probably the MIT license, and provided on GitHub.
